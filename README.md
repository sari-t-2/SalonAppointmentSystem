# 2SBA Salon & Spa Scheduler

A console-based Java application built using Apache NetBeans IDE. This project acts as an automated appointment management utility for small-scale boutique salons.

## Features
- Book new client appointments with auto-pricing.
- Prevent duplicate time-slot bookings.
- View, reschedule, or cancel active sessions.

## How to Run
1. Open the project folder inside **Apache NetBeans IDE**.
2. Right-click the project and select **Clean and Build**.
3. Press the green **Run** icon to launch the application console.
